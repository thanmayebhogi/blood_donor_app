# 🩸 BloodDonor App

An Android application to connect blood donors and receivers in emergency situations. Built to make donor search fast, simple, and location-based.

### ✨ Features
- 🔐 Secure User Authentication (Firebase Auth)
- 📝 Donor Registration & Profile Management
- 🔍 Search by Blood Group (A+, B+, O+, AB+, etc.)
- 🚨 Emergency Donor Search
- 📍 Location-Based Donor Connectivity
- 📞 Direct Call to Donor
- ⏱️ Real-time Availability Status (Firebase Realtime Database)

### 🛠️ Tech Stack
- **Language:** Java
- **UI:** XML, Material Design
- **Backend:** Firebase Authentication, Firebase Realtime Database
- **Tools:** Android Studio, Git

### 🚀 How to Run
1. Clone the repo
git clone https://github.com/thanmayebhogi/BloodDonor-App-2026.git
2. Open in Android Studio
3. Add your own `google-services.json` file from Firebase Console to `app/` folder
4. Sync Gradle and Run

### 🔒 Note
`google-services.json` is not included for security reasons. Create your own Firebase project to test.
